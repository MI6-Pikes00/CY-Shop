## Instructions

1. Open a terminal window 

2. Clone the git repository to a location of your choice   
```sh
   git clone https://github.com/MI6-Pikes00/Projet-preing1
   ```

### Compiler avec Make
3. Enter the new folder
   ```sh
   cd Projet-preing1
   ```
4. Compile and run the project with *make* :
   ```sh
   make && ./CY-Shop
   ```
